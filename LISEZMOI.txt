Karst Entry 1.0
=========================

Plugin QGIS de saisie de phenomenes karstiques.

INSTALLATION
------------
Windows : double-cliquer sur install_plugin.bat
Linux   : chmod +x install_plugin.sh puis ./install_plugin.sh

Le script copie le dossier karst_entry/ dans le profil QGIS detecte
(QGIS 3 et/ou QGIS 4). Activez ensuite l'extension dans QGIS :
Extensions > Gerer et installer les extensions > Installees > Karst Entry.

Installation manuelle (alternative) : copier le dossier karst_entry/ dans
  Windows : %APPDATA%\QGIS\QGIS4\profiles\default\python\plugins\
  Linux   : ~/.local/share/QGIS/QGIS4/profiles/default/python/plugins/
